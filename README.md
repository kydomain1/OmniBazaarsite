# OmniBazaarsite - Lifestyle Blog Website

A beautiful, modern lifestyle blog website featuring soft, sweet design aesthetics with Morandi color palette, rounded corners, and smooth animations.

## Features

- **6 Main Categories:**
  - Fashion & Accessories
  - Health & Beauty
  - Home & Garden
  - Travel & Accommodation
  - Finance & Insurance
  - Food & Beverages

- **Key Functionality:**
  - Blog article listing with pagination
  - Category filtering
  - Search functionality
  - Individual article pages
  - Product recommendations
  - About page
  - Contact page
  - Responsive mobile design
  - Social media integration
  - Smooth animations and transitions

## Design Features

- **Soft & Sweet Style:**
  - Low saturation Morandi color palette
  - Rounded corners throughout
  - Smooth animations and transitions
  - Warm, comfortable, friendly feel

## File Structure

```
OmniBazaarsite/
├── index.html          # Main homepage
├── article.html        # Article detail page
├── about.html          # About page
├── contact.html        # Contact page
├── styles.css          # Main stylesheet
├── script.js           # JavaScript functionality
├── data.js             # Blog posts data
└── README.md           # This file
```

## Usage

1. Open `index.html` in a web browser
2. Browse articles by category or use the search function
3. Click on any article to view the full content
4. Navigate to About or Contact pages using the header menu

## Customization

- Edit `data.js` to add or modify blog posts
- Modify `styles.css` to change colors, fonts, or layout
- Update `script.js` to add new functionality

## Browser Support

Works on all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge

## Notes

- This is a static website (no server required)
- Images are loaded from Unsplash (requires internet connection)
- All functionality is client-side JavaScript
